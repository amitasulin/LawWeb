# פתרון בעיות - האתר לא עולה

## ✅ השרת רץ בהצלחה!

אם השרת רץ אבל אתה לא רואה את האתר, נסה את הפתרונות הבאים:

### 1. בדוק את הכתובת
פתח את הדפדפן ונסה:
- **http://localhost:3000**
- או: **http://127.0.0.1:3000**

### 2. בדוק אם השרת רץ
בטרמינל אמור לראות:
```
✓ Ready in X.Xs
- Local: http://localhost:3000
```

אם לא רואה - השרת לא רץ. נסה:
```bash
cd web
npm run dev
```

### 3. בדוק את הקונסול של הדפדפן
1. לחץ **F12** בדפדפן
2. לך לטאב **Console**
3. חפש שגיאות (אדומות)

### 4. נסה דפדפן אחר
- Chrome
- Firefox
- Edge

### 5. נקה את ה-Cache
- לחץ **Ctrl+Shift+R** (או Cmd+Shift+R ב-Mac)
- או: **Ctrl+F5**

### 6. בדוק אם יש דבר אחר על פורט 3000
```bash
netstat -ano | findstr :3000
```

אם יש - הרוג את התהליך:
```bash
taskkill /PID <מספר_PID> /F
```

### 7. נסה פורט אחר
אם פורט 3000 תפוס, שנה את הפורט:
```bash
npm run dev -- -p 3001
```
ואז לך ל: http://localhost:3001

---

## אם כלום לא עובד:

1. **עצור את השרת** (Ctrl+C)
2. **מחק את התיקייה `.next`**:
   ```bash
   rm -rf .next
   # או ב-Windows:
   rmdir /s /q .next
   ```
3. **הפעל מחדש**:
   ```bash
   npm run dev
   ```
